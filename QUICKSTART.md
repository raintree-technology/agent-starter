# Quick Start Guide

## The Simple, Useful Thing: Zero-Config Claude Skills

This package lets users instantly add Claude Code skills to their project in **30 seconds** with zero configuration.

---

## User Experience

### Option 1: Interactive (Recommended)

```bash
npx create-claude-starter
```

**Prompts:**
```
🚀 Claude Starter

? Which skills do you want to install?
  ❯ Web SaaS Starter - Payments + Backend + Mobile
    Blockchain Developer - Aptos, Move, Shelby
    AI Developer - Anthropic API, Claude Code skills
    Mobile Developer - Expo, React Native, iOS
    Fintech Stack - Payments, banking APIs
    Minimal (TOON Only) - Just token optimization
    All Skills - 40+ skills across all categories
    Custom Selection - Pick exactly what you want

[User picks "Web SaaS Starter"]

📦 Installing claude-starter...
  ✓ Installed profile: Web SaaS Starter (4 skills)
  ✓ Generated settings.json
  ✓ Setup TOON binary (darwin-arm64)

✅ Claude starter installed successfully!

Next steps:
  1. Pull documentation: npx claude-starter docs pull stripe
  2. List available skills: npx claude-starter list
```

### Option 2: Profile Flags

```bash
# Web SaaS (Stripe, Supabase, Expo)
npx create-claude-starter --profile web-saas

# Blockchain (Aptos, Shelby, Decibel)
npx create-claude-starter --profile blockchain

# AI Developer (Anthropic, Claude Code)
npx create-claude-starter --profile ai-developer

# Just TOON optimization
npx create-claude-starter --profile minimal

# Everything
npx create-claude-starter --profile all
```

### Option 3: Specific Skills

```bash
# Pick exactly what you want
npx create-claude-starter --skills stripe,supabase,react
```

### Option 4: Silent Install

```bash
# No prompts, use defaults
npx create-claude-starter --yes
```

## What Gets Installed

### Directory Structure

```
your-project/
├── .claude/
│   ├── skills/
│   │   ├── stripe/skill.md
│   │   ├── supabase/skill.md
│   │   └── ...
│   ├── commands/
│   │   ├── analyze-tokens.md
│   │   ├── convert-to-toon.md
│   │   └── ...
│   ├── utils/toon/
│   │   ├── bin/toon (native binary)
│   │   └── toon-guide.md
│   └── settings.json (auto-generated)
└── ...
```

### settings.json (Auto-Generated)

```json
{
  "$schema": "https://code.claude.com/schemas/settings.schema.json",
  "version": "1.0.0",
  "description": "Claude Code configuration - auto-generated by create-claude-starter",
  "skills": {
    "stripe": {
      "enabled": true,
      "model": "sonnet"
    },
    "supabase": {
      "enabled": true,
      "model": "sonnet"
    }
  },
  "toon": {
    "enabled": true,
    "binaryPath": ".claude/utils/toon/bin",
    "autoDetect": true
  }
}
```

**Skills are auto-activated** - just mention "Stripe" or "Supabase" in conversation with Claude and the skill activates.

## Commands Available

After installation, you get:

```bash
# Add more skills later
npx claude-starter add stripe supabase

# List all available skills
npx claude-starter list

# List by category
npx claude-starter list --category payments

# Update installed skills
npx claude-starter update

# Manage documentation
npx claude-starter docs pull stripe      # Pull Stripe docs (3,253 files)
npx claude-starter docs pull             # Pull all installed skills' docs
npx claude-starter docs status           # Check what's installed
npx claude-starter docs update           # Update stale docs
```

## How Skills Work

Skills **auto-activate** based on keywords in your conversation:

```
You: "How do I create a Stripe subscription?"
→ Stripe skill activates automatically

You: "Set up Supabase authentication"
→ Supabase skill activates automatically

You: "This JSON file is too big"
→ TOON formatter skill activates automatically
```

No manual invocation needed!

## Documentation (Optional)

Skills work immediately with **built-in knowledge**. Want comprehensive API docs?

```bash
# Install docpull
brew install pipx
pipx install docpull

# Pull docs for specific skills
npx claude-starter docs pull stripe      # 3,253 files, 33MB
npx claude-starter docs pull supabase    # 2,616 files, 111MB

# Or pull all at once
npx claude-starter docs pull
```
